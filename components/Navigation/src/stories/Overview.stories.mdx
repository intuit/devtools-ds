## Usage

The `Navigation` component allows you to construct a navigation bar with tabs, buttons, and dividers that help users navigate your devtools application.
In addition, the component manages showing and hiding different "panels" for you.
This helps with accessibility, so we can automatically link the `id` of a tab to its corresponding panel.
