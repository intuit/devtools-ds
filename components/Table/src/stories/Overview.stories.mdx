## Usage

The `Table` component is an HTML Table styled after the network inspector in Chrome and Firefox.

<Canvas>
  <Story id="components-table-features--basic-usage" />
</Canvas>

### Selecting Rows

You can use the `selected` and `onSelected` props to track which row is selected.
Adding these props will enable hover and focus styles.

<Canvas>
  <Story id="components-table-features--selected-row" />
</Canvas>

### Resizing

Each column is user-resizable, and the `Table` will automatically fill the space of its container as needed.
